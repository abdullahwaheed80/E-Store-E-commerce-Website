# E-Store-E-commerce-Website

Project Overview:
This is my second project as part of my Web Development journey at METEC Vision. After learning the basics of HTML and CSS, this project was my introduction to using the Bootstrap 5 framework to create a responsive, mobile-friendly e-commerce interface.

Features & Learning Milestones:
Bootstrap Grid System: Implemented a responsive product grid that adjusts from desktop to mobile views.
Hero Section: Created a themed showcase area using custom CSS and Bootstrap classes.
Navigation & Footer: Developed a functional navbar and a multi-column footer layout.
Form Integration: Basic sign-up/newsletter section with Bootstrap styling.

Tech Stack:
HTML5 (Structure)
CSS3 (Custom Styling & Positioning)
Bootstrap 5 (Layout & Responsiveness)
